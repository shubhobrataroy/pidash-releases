# PiDash Desktop Releases

Latest desktop builds for PiDash are available below:

## Available Downloads

| Platform | Format | File |
|----------|--------|------|
| macOS | DMG Installer | `PiDash-*-macOS.dmg` |
| Windows | MSI Installer | `PiDash-*-windows.msi` |
| Linux | AppImage | `PiDash-*-linux.AppImage` |

## Installation Instructions

### macOS
1. Download the `.dmg` file
2. Open the DMG file
3. Drag PiDash to Applications folder

### Windows
1. Download the `.msi` file
2. Run the installer
3. Follow the installation wizard

### Linux
1. Download the `.AppImage` file
2. Make it executable: `chmod +x PiDash-*.AppImage`
3. Run: `./PiDash-*.AppImage`
